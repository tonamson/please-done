    markdownToHtml(md) — regex-based converter (per RESEARCH recommendation, zero deps):
    1. Mermaid code blocks: /```mermaid\n([\s\S]*?)```/g -> `<pre class="mermaid">$1</pre>`
    2. Other code blocks: /```(\w*)\n([\s\S]*?)```/g -> `<pre><code>$2</code></pre>`
    3. Horizontal rules: /^---$/gm -> `<hr>`
    4. Headings h6->h1 (process h6 first to avoid h1 matching ##): /^#{N}\s+(.+)$/gm
    5. Bold: /\*\*(.+?)\*\*/g -> `<strong>$1</strong>`
    6. Italic: /\*(.+?)\*/g -> `<em>$1</em>`
    7. Tables: parse line-by-line — detect `|` rows, skip alignment row `|---|`, build `<table><thead><tr><th>...</th></tr></thead><tbody><tr><td>...</td></tr></tbody></table>`
    8. Blockquotes: /^>\s+(.+)$/gm -> `<blockquote>$1</blockquote>`
    9. Unordered lists: /^- (.+)$/gm -> `<li>$1</li>`, then wrap consecutive `<li>` in `<ul>`

    buildHtml(bodyHtml) — wraps body in full HTML document:
    - DOCTYPE html, html lang="vi", head with meta charset UTF-8
    - style tag containing CSS_TEMPLATE (per D-05: inline CSS, no separate file)
    - body containing bodyHtml
    - script type="module" importing mermaid from CDN: `https://cdn.jsdelivr.net/npm/mermaid@11/dist/mermaid.esm.min.mjs` (per D-01, D-03)
    - mermaid.initialize({ startOnLoad: true, theme: 'default' }) (per D-01)

    canUsePuppeteer() — runtime detection (per D-07, D-08, D-10):
    - Check parseInt(process.versions.node, 10) < 18 -> { available: false, reason: 'Node.js X < 18. Can Node >= 18 de render PDF.' }
    - try require.resolve('puppeteer') -> { available: true }
    - catch -> { available: false, reason: 'PDF export can Puppeteer. Chay: npm install puppeteer' }

    renderMarkdownFallback(markdownContent) — returns markdownContent unchanged (per D-09: fallback outputs filled Markdown).

    module.exports = { markdownToHtml, buildHtml, CSS_TEMPLATE, canUsePuppeteer, renderMarkdownFallback };
