## Translation Pattern (từ cheatsheet.vi.md)

### Header Comment Pattern
```markdown
<!-- Translated from docs/skills/{file}.md -->
<!-- Source version: 4.0.0 -->
<!-- Translation date: 2026-04-05 -->
```

### Language Switcher Badges (đặt ngay sau header comments)
```markdown
[![English](https://img.shields.io/badge/lang-English-blue.svg)]({file}.md)
[![Tiếng Việt](https://img.shields.io/badge/lang-Tiếng%20Việt-red.svg)]({file}.vi.md)
```

### Section Header Translations
| English | Vietnamese |
|---------|------------|
| Purpose | Mục đích |
| When to Use | Khi nào dùng |
| Prerequisites | Điều kiện tiên quyết |
| Basic Command | Lệnh cơ bản |
| Common Flags | Các cờ phổ biến |
| See Also | Xem thêm |
| What it does | Chức năng |
| What it creates | Nội dung được tạo |

### Rules (per D-01, D-02, D-03)
- DỊCH: Mô tả, giải thích, hướng dẫn
- GIỮ NGUYÊN: Commands (/pd:xxx), flags (--auto), file paths, code examples, skill names trong links
- Skill names trong content: giữ English (e.g., "onboard skill", "init command")
