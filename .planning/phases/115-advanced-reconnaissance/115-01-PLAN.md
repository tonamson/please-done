## Class: WorkflowMapper

Constructor: `constructor(options = {})` with `this.cache` (ReconCache), `this.workflows`, `this.flaws`.

## Method: analyze(filePath)
1. Read file via `fs.promises.readFile`
2. Parse AST using `@babel/parser` with same plugins as source-mapper.js (lines 61-73)
3. Call `detectStateMachine(ast, filePath)`
4. Call `detectBusinessLogicFlaws(ast, filePath)`
5. Return `{ workflows: { states, transitions }, flaws: [], filePath }`

## Method: detectStateMachine(ast, filePath)
Traverse AST with @babel/traverse. For each CallExpression where callee is MemberExpression with property in ['get','post','put','delete','patch']:
1. Extract route path from first argument (args[0]?.value)
2. Extract middleware chain from remaining arguments (args.slice(1))
3. For each adjacent pair in middleware chain, create transition: `{ from: handler[i], to: handler[i+1], trigger: METHOD route, type: classifyTransition(handler[i]) }`
4. Track states: each unique handler is a state node
5. Return `{ states: [{ id, name, type }], transitions: [{ from, to, trigger, type }] }`

State types: 'guard' (checks auth/validation), 'action' (mutations/DB writes), 'transition' (navigation/redirects)

## Method: detectBusinessLogicFlaws(ast, filePath)

```javascript
/**
 * Run workflow analysis on multiple files
 * Phase 115: RECON-06 Business Logic Mapping
 */
async runWorkflowAnalysis(files) {
  const allWorkflows = [];
  const allFlaws = [];

  // Limit to 50 files for expensive AST analysis (per RESEARCH.md Pitfall 4)
  const analysisFiles = files.slice(0, 50);

  for (const file of analysisFiles) {
    try {
      const result = await this.workflowMapper.analyze(file);
      if (result.workflows.states.length > 0) {
        allWorkflows.push(result);
      }
      if (result.flaws.length > 0) {
        allFlaws.push(...result.flaws);
      }
    } catch (err) {
      continue;
    }
  }

  return {
    workflows: allWorkflows,
    flaws: allFlaws,
    summary: {
      filesAnalyzed: analysisFiles.length,
      workflowsDetected: allWorkflows.length,
      flawsFound: allFlaws.length,
      criticalFlaws: allFlaws.filter(f => f.severity === 'CRITICAL').length,
      highFlaws: allFlaws.filter(f => f.severity === 'HIGH').length
    }
  };
}
```
