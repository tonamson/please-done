# Plan 147-02: Wire classifier into install.js + write smoke tests

## Goal
Three concrete edits: (A) remove the pre-throw `log.error` from `install()`'s Step 2 catch, (B) replace `main().catch()` body with classifier-based output, (C) create `test/smoke-errors.test.js` covering all 4 categories.

## Context
- **D-07**: Output format is `log.error(classified.category + ': ' + classified.message)` + `log.info('  Hint: ' + classified.hint)` — `log.error` prepends `  ✗ `, category name must be explicitly prefixed so users see e.g. `  ✗ PERMISSION: EACCES: ...`
- **D-08**: `main().catch()` uses classifier; stack only with `PD_DEBUG=1`
- **D-09**: `throw err` in `install()` catch stays — only the `log.error` line above it is removed
- **D-10**: 4 test cases in `test/smoke-errors.test.js`, pattern matches `test/smoke-install.test.js`
- **D-11**: No changes to individual installer throw messages
- Research finding: `main().catch()` currently has `log.error(err.message)` (line 380) — replace entire catch body

### Task 2: Update main().catch() in bin/install.js

**File(s)**: `bin/install.js`

**Action**:

Locate the `main().catch()` block at the bottom of the file (around line 379–382). It currently reads:

```js
  main().catch((err) => {
    log.error(err.message);
    if (process.env.PD_DEBUG) console.error(err.stack);
    process.exit(1);
  });
```

Replace the entire catch body with classifier-based output per D-07 and D-08:

```js
  main().catch((err) => {
    const { classifyError } = require('./lib/error-classifier');
    const classified = classifyError(err);
    log.error(classified.category + ': ' + classified.message);
    log.info('  Hint: ' + classified.hint);
    if (process.env.PD_DEBUG) console.error(err.stack);
    process.exit(1);
  });
```

The `require` is intentionally inline inside the catch callback (lazy load — only loaded on failure path). The `if (process.env.PD_DEBUG)` line is kept to satisfy D-08.

**Expected result**: `grep -n "classifyError" bin/install.js` returns a match. `grep -n "log.error(err.message)" bin/install.js` returns no output.

### Task 3: Create test/smoke-errors.test.js

**File(s)**: `test/smoke-errors.test.js`

**Action**:

Create the file following the exact pattern from `test/smoke-install.test.js` (uses `node:test`, `node:assert/strict`, `describe`/`it` structure, `'use strict'`, no `PD_TEST_MODE` needed).

Implement 4 test cases covering every category:

**Test 1 — PERMISSION (with path)**:
- Input: `Object.assign(new Error('EACCES: permission denied, open \'/home/user/.claude\''), { code: 'EACCES', path: '/home/user/.claude' })`
- Assert: `result.category === 'PERMISSION'`
- Assert: `result.hint.includes('sudo chown')`
- Assert: `result.hint.includes('/home/user/.claude')`

**Test 2 — MISSING_DEP (with URL in message)**:
- Input: `new Error('Python not installed. Install first: https://python.org/downloads')`
- Assert: `result.category === 'MISSING_DEP'`
- Assert: `result.hint === 'https://python.org/downloads'`

**Test 3 — PLATFORM_UNSUPPORTED**:
- Input: `Object.assign(new Error("Cannot find module './lib/installers/windows'"), { code: 'MODULE_NOT_FOUND' })`
- Assert: `result.category === 'PLATFORM_UNSUPPORTED'`
- Assert: `result.hint.includes('not yet supported')`

**Test 4 — GENERIC**:
- Input: `new Error('Unexpected internal error')`
- Assert: `result.category === 'GENERIC'`
- Assert: `result.message === 'Unexpected internal error'`

**Full file to produce:**

```js
/**
 * Smoke tests for bin/lib/error-classifier.js
 * Phase 147 — D-10
 * Run: node --test test/smoke-errors.test.js
 */

'use strict';

const { describe, it } = require('node:test');
const assert = require('node:assert/strict');
const { classifyError } = require('../bin/lib/error-classifier');

describe('classifyError', () => {
  it('PERMISSION: EACCES with path → category PERMISSION, hint has sudo chown + path', () => {
    const err = Object.assign(
      new Error("EACCES: permission denied, open '/home/user/.claude'"),
      { code: 'EACCES', path: '/home/user/.claude' }
    );
    const result = classifyError(err);
    assert.equal(result.category, 'PERMISSION');
    assert.ok(result.hint.includes('sudo chown'), `hint should include 'sudo chown', got: ${result.hint}`);
    assert.ok(result.hint.includes('/home/user/.claude'), `hint should include path, got: ${result.hint}`);
  });

  it('MISSING_DEP: message with URL → category MISSING_DEP, hint is the URL', () => {
    const err = new Error('Python not installed. Install first: https://python.org/downloads');
    const result = classifyError(err);
    assert.equal(result.category, 'MISSING_DEP');
    assert.equal(result.hint, 'https://python.org/downloads');
  });

  it('PLATFORM_UNSUPPORTED: MODULE_NOT_FOUND code → category PLATFORM_UNSUPPORTED, hint has not yet supported', () => {
    const err = Object.assign(
      new Error("Cannot find module './lib/installers/windows'"),
      { code: 'MODULE_NOT_FOUND' }
    );
    const result = classifyError(err);
    assert.equal(result.category, 'PLATFORM_UNSUPPORTED');
    assert.ok(result.hint.includes('not yet supported'), `hint should include 'not yet supported', got: ${result.hint}`);
  });

  it('GENERIC: unrecognised error → category GENERIC, message passed through', () => {
    const err = new Error('Unexpected internal error');
    const result = classifyError(err);
    assert.equal(result.category, 'GENERIC');
    assert.equal(result.message, 'Unexpected internal error');
  });
});
```

**Expected result**: `node --test test/smoke-errors.test.js` exits 0 with output showing 4 passing tests.
