   - `htmlEncode/htmlDecode` - HTML entity encoding
   - `unicodeEncode/unicodeNormalize` - Unicode escape sequences and normalization

2. **Command Obfuscation** (PAYLOAD-02, T1027.010):
   - `caseObfuscate(str)` - Mixed case variant (e.g., "cAt" for "cat")
   - `commentInject(payload)` - Inject inline comments between characters to break payload signatures (e.g., "c/**/a/**/t")
   - `reverseCommand(cmd)` - Reverse command string
   - `nullByteInject(cmd)` - Prepend null bytes

3. **Multi-Layer Encoding** (PAYLOAD-03):
   - `multiLayerEncode(payload, layers)` - Apply multiple encoding layers in sequence
   - `generateEncodedVariants(payload, attackType)` - Generate all variants for a payload type

4. **XSS/SQLi Evasion** (PAYLOAD-04):
   - `generateXssVariants(payload)` - XSS evasion using case variation, comment injection, Unicode encoding
   - `generateSqliVariants(payload)` - SQLi evasion using comment injection, null bytes, encoding

5. **Double Extension Detection** (PAYLOAD-05, T1036.007):
   - `detectDoubleExtension(filename)` - Analyze filename for double extension patterns
   - Patterns: .php.jpg, .asp.txt, .js.php, .exe.png, etc.

6. **WAF Profile Configuration** (PAYLOAD-01):
   - Pre-configured profiles for major WAFs: Cloudflare, ModSecurity, Akamai, AWS WAF
   - Each profile specifies known bypass patterns
