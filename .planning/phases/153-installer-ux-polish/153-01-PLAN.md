### Task 3: Improve Progress Step Formatting

**File:** `bin/lib/utils.js`

**Action:** Enhance the `log.step()` function to use a more visually distinct format with a progress indicator.

**Details:**

Find this line in the `log` object:
```js
  step: (num, total, msg) =>
    console.log(colorize("cyan", `[${num}/${total}] ${msg}`)),
```

Replace with:
```js
  step: (num, total, msg) => {
    const progress = "●".repeat(num) + "○".repeat(total - num);
    const isTTY = process.stdout.isTTY && !process.env.NO_COLOR;
    const prefix = isTTY
      ? `${COLORS.cyan}[${num}/${total}]${COLORS.reset} ${COLORS.dim}${progress}${COLORS.reset}`
      : `[${num}/${total}] ${progress}`;
    console.log(`${prefix}  ${msg}`);
  },
```

This changes the step output from:
```
[1/4] Backing up locally modified files...
```

To:
```
[1/4] ●○○○  Backing up locally modified files...
```
