## Task 1: Create Source Mapper Library

**Goal:** Implement SourceMapper class for identifying untrusted data sources

**Steps:**
1. Create `bin/lib/source-mapper.js` with:
   - AST parser using @babel/parser with TypeScript support
   - Source identification patterns (req.body, req.query, req.params, headers, cookies, files)
   - Sink mapping (SQL queries, eval, exec, innerHTML, etc.)
   - Taint tracking through variable assignments
   - Integration with ReconCache from Phase 112

2. Implement methods:
   - `analyze(filePath)` - Parse file and identify sources/sinks
   - `getSources()` - Return array of identified untrusted sources
   - `getSourceToSinkMap()` - Return mapping of sources to their sinks
   - `getRiskyFlows()` - Return high-risk data flows (source -> dangerous sink)

3. Output format:
   ```javascript
   {
     source: {
       type: 'req.body',
       location: { file: 'controllers/user.js', line: 15, column: 10 },
       variable: 'userInput'
     },
     sinks: [
       {
         type: 'sql.query',
         location: { file: 'services/db.js', line: 42, column: 5 },
         risk: 'high'
       }
     ]
   }
   ```

**Commit:** `feat(phase-113): implement source mapper for untrusted data detection`

## Task 2: Create Target Enumerator Library

**Goal:** Implement TargetEnumerator class for API endpoint discovery

**Steps:**
1. Create `bin/lib/target-enumerator.js` with:
   - Framework detection (Express, Fastify, NestJS, Next.js, Koa, Hapi)
   - Route extraction from AST patterns
   - Middleware chain analysis
   - Authentication requirement detection
   - Hidden/undocumented endpoint detection

2. Implement methods:
   - `discoverRoutes(filePath)` - Extract route definitions from file
   - `analyzeMiddleware(route)` - Identify middleware applied to route
   - `checkAuthRequirement(route)` - Determine if route requires authentication
   - `getApiSpecification()` - Generate OpenAPI-like spec of discovered routes

3. Route detection patterns:
   - Express: `app.get('/path', handler)`, `router.post('/path', handler)`
   - Fastify: `fastify.get('/path', handler)`
   - NestJS: Decorator analysis `@Get('/path')`
   - Next.js: File-based route scanning
   - Koa: `@koa/router` patterns
   - Hapi: `server.route({ method, path })`

4. Output format:
   ```javascript
   {
     path: '/api/users/:id',
     methods: ['GET', 'POST'],
     file: 'routes/users.js',
     line: 23,
     middleware: ['auth', 'validate'],
     authRequired: true,
     isDocumented: true
   }
   ```

**Commit:** `feat(phase-113): implement target enumerator for API discovery`
