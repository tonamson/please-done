## Tasks

### Task 1: Tao research-store.js voi createEntry va parseEntry
type="auto"

Tao file `bin/lib/research-store.js` voi cac pure functions:

1. **CONFIDENCE_LEVELS** constant — { HIGH, MEDIUM, LOW } voi mo ta tieu chi
2. **REQUIRED_FIELDS** constant — danh sach truong bat buoc trong frontmatter (agent, created, source, topic, confidence)
3. **createEntry(options)** — tao markdown content voi frontmatter chuan
   - options: { agent, source ('internal'|'external'), topic, confidence, body, id?, slug? }
   - Voi external: ten file theo format `RES-[ID]-[SLUG].md`
   - Tu dong them created: ISO-8601
   - Tra ve { content, filename }
4. **parseEntry(content)** — parse research file content
   - Dung parseFrontmatter tu utils.js
   - Validate required fields
   - Tra ve { frontmatter, body, valid, errors[] }
5. **validateConfidence(level)** — kiem tra confidence hop le (HIGH/MEDIUM/LOW)
6. **generateFilename(options)** — tao ten file theo source type

Done: Module export 4+ functions, KHONG doc file, KHONG require('fs').
