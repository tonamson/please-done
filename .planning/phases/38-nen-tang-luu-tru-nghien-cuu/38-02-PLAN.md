3. **createEntry({ source, topic, content, agent, confidence, tags, scope, existingFiles })** — per D-07:
   - Validate required: topic (string, non-empty), agent (string, non-empty), source (in VALID_SOURCES), confidence (in VALID_CONFIDENCE)
   - Throw Error voi message tieng Viet khi validation fail: 'thieu tham so topic', 'thieu tham so agent', 'confidence khong hop le: {value}'
   - Tao filename: external -> `${nextResId(existingFiles)}-${generateSlug(topic)}.md`, internal -> `${generateSlug(topic).toUpperCase().replace(/-/g, '_')}.md` per D-11

   - Tao frontmatter object 8 fields per D-01: agent, created (new Date().toISOString()), source, topic, confidence, scope, expires (30 ngay sau), tags
   - Tao body: `\n# ${source === 'external' ? id + ': ' : ''}${topic}\n\n${content}\n`
   - Content da truyen vao PHAI chua inline confidence markers (caller chiu trach nhiem). createEntry giu nguyen content, KHONG tu dong them/xoa markers.
   - Dung assembleMd(frontmatter, body) de tao content
   - Return { filePath, fileName, frontmatter, content: fileContent }

4. **parseEntry(content)** — per D-07 + AUDIT-03:
