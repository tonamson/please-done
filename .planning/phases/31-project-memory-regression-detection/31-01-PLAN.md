    createBugRecord({ existingBugs, file, functionName, errorMessage, sessionId, rootCause, fix }):
    - Validate: file va rootCause bat buoc (throw Error neu thieu)
    - nextBugNumber: dung pattern giong session-manager.js — scan existingBugs lay max number + 1, default 1 neu rong
    - id = `BUG-${String(num).padStart(3, '0')}` (per D-01)
    - fileName = `${id}.md`
    - frontmatter object: { file: file.trim(), function: (functionName||'').trim(), error_message: (errorMessage||'').trim(), session_id: (sessionId||'').trim(), resolved_date: YYYY-MM-DD, status: 'resolved' } (per D-02)
    - body = `\nNguyen nhan: ${rootCause.trim()}\nFix: ${fix ? fix.trim() : ''}\n` (per D-02: 1-2 dong, khong bloat)
    - content = assembleMd(frontmatter, body)
    - Return { id, fileName, content, number: num }

    searchBugs({ bugRecords, file, functionName, errorMessage }):
    - Guard: neu ca 3 tieu chi deu rong/falsy → return { matches: [], warnings: ['thieu tat ca tieu chi tim kiem'] }
    - Loop qua bugRecords:
      + Neu record khong co frontmatter → push warning 'bug record khong co frontmatter', continue
      + score = 0
      + File path (D-05): case-insensitive substring CA 2 CHIEU — f1.includes(f2) || f2.includes(f1). Score++ neu khop
      + Function (D-05): case-insensitive EXACT match — func1.toLowerCase() === func2.toLowerCase(). Score++ neu khop
      + Error message (D-05): case-insensitive substring CA 2 CHIEU. Score++ neu khop
      + Neu score >= 1: push vao matches { id: record.id || 'UNKNOWN', score, file: fm.file || '', function: fm.function || '', error_message: fm.error_message || '' }
    - Sort matches by score DESC
    - Return { matches, warnings }

    buildIndex(bugRecords):
    - Neu bugRecords rong: return header voi "Tong so: 0 bugs"

      + "## Tat ca Bugs" — table | ID | File | Function | Error | Status | Session | Resolved |
    - Header: "# Bug Index\n\n**Cap nhat:** {ISO timestamp}\n**Tong so:** {N} bugs"
    - Return complete markdown string
