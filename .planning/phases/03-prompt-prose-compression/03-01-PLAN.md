<task type="auto">
  <name>Task 1: Install js-tiktoken and create token counting script</name>
  <files>package.json, scripts/count-tokens.js</files>
  <action>
1. Install js-tiktoken as devDependency:
   ```
   npm install --save-dev js-tiktoken
   ```

2. Create `scripts/count-tokens.js` that:
   - Uses `encodingForModel('gpt-4o')` from js-tiktoken (cl100k_base encoding)
   - Scans 4 directories: `commands/pd/`, `workflows/`, `references/`, `templates/`
   - For each .md file: reads content, counts tokens via `enc.encode(content).length`, counts lines
   - Skips `references/guard-*.md` files (1-line micro-templates, already minimal)
   - Outputs a table to stdout: `file | tokens | lines`
   - Groups by directory with subtotals
   - Shows grand total at the end
   - Supports `--baseline` flag: writes results to `test/baseline-tokens.json` as `{ files: { [relativePath]: { tokens, lines } }, total: N, capturedAt: ISO timestamp }`
   - Supports `--compare` flag: reads `test/baseline-tokens.json`, compares current counts vs baseline, shows per-file delta and percentage change, shows total delta and percentage
   - Uses only `node:fs`, `node:path`, and `js-tiktoken` (no other dependencies)
   - Uses CommonJS require (project convention -- no ESM)
   - Exit code 0 on success, 1 on error
  </action>
  <verify>
    <automated>node scripts/count-tokens.js</automated>
  </verify>
  <done>Script runs without error, prints token counts for all target files grouped by directory with totals</done>
</task>
