  <action>
Create bin/lib/asset-discoverer.js implementing the AssetDiscoverer class with these methods:

1. Constructor accepting options with cache parameter (default to new ReconCache())
2. discoverHiddenAssets(projectPath, options) - main entry point returning Promise
3. loadWordlist(name, tier) - loads wordlist files from references/wordlists/
4. scanFileSystem(projectPath, patterns) - scans filesystem for backup/config files
5. classifyAssetType(path) - returns one of: 'admin-panel', 'debug-endpoint', 'config-exposure', 'backup-file', 'source-map', 'unknown'
6. calculateRisk(asset) - returns severity per D-13 to D-16:
   - Critical: admin-panel without auth, debug-endpoint in production, backup-file exposed, hardcoded credentials found
   - High: config-exposure, source-map, weak JWT
   - Medium: undocumented API, test pages
   - Low: dev routes, internal docs

Risk calculation rules:
- If type is 'admin-panel' and requiresAuth is false, severity = 'CRITICAL'
- If type is 'debug-endpoint', severity = 'HIGH'
- If type is 'config-exposure', severity = 'HIGH'
- If type is 'backup-file', severity = 'HIGH'
- If type is 'source-map', severity = 'MEDIUM'
- Default severity = 'LOW'

Export: { AssetDiscoverer }
  </action>
